# Wastra Digital

Marketplace digital untuk mempromosikan dan memasarkan kain tradisional (endek & songket) dari Desa Sidemen, Karangasem, Bali.

## Teknologi

- **React.js** - Library UI
- **Vite** - Build tool
- **Ant Design** - Komponen UI untuk form, tabel, modal
- **Tailwind CSS** - Utility-first CSS untuk layout & spacing
- **Styled Components** - CSS-in-JS untuk komponen custom
- **Heroicons** - Icon library
- **React Router** - Routing

## Instalasi

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Struktur Folder

```
src/
├── components/     # Komponen reusable
├── pages/         # Halaman aplikasi
├── assets/        # Gambar, font, dll
├── utils/         # Utility functions
├── styles/        # Global styles
└── App.jsx        # Root component
```

## Catatan

- Semua teks UI dalam Bahasa Indonesia
- Tailwind untuk layout & spacing
- Ant Design untuk form, tabel, modal
- Styled Components hanya untuk komponen custom
- Preflight Tailwind dinonaktifkan untuk menghindari konflik dengan Ant Design
